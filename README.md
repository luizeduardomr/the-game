## Run Locally

Clone the project

```bash
  git clone https://github.com/luizeduardomr/the-game/
```

Go to the project directory

```bash
  cd the-game
```

Open the XCode Project (open the .xcworkspace file)
```bash
open the-game.xcworkspace
```

Run the project
```bash
  And start the simulator =)
```
